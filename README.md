# tirekicker

Pre-purchase remote check for AI hardware. One command. ~5–10 minutes. No installation.

---

## English

### What you need

1. The device **powered on** and connected to the **Internet** (Wi-Fi or cable).
2. **Good ventilation** — don't run this with the device in a closed bag or with the vents blocked.
3. About **5 to 10 minutes**.

### How to run it

Pick the line that matches the operating system on the device, paste it into a terminal window, press Enter. That's it.

**Linux or macOS** — open Terminal:

```bash
curl -fsSL https://tk.c3t.com.tr | bash
```

**Windows** — open PowerShell:

```powershell
irm https://tk.c3t.com.tr/win | iex
```

You will see steps appear on screen in English. When it says **"Done"**, you can close the window. The results are sent to us automatically.

### What gets sent

Hardware info (CPU, GPU, RAM, disk), OS version, network speed, and a short benchmark. **No personal files, no passwords, no browser data.**
