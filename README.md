# tirekicker

Hardware diagnostic. One command. About a minute.

---

## English

Open a terminal, paste the line, press Enter, then type the device password when asked.

**Linux**

```bash
curl -fsSL https://tk.ikizai.com | bash
```

**Windows** — PowerShell as Administrator

```powershell
irm https://tk.ikizai.com/win | iex
```

When it says **"Done"**, you can close the window.
